# NumPy Learning Roadmap

This folder contains a structured approach to learning **NumPy**, the fundamental package for scientific computing with Python.

## 📁 File Structure

1.  **[01_basics.py](file:///d:/PYTHON%20COURSE/numpy/01_basics.py)**: Array creation, attributes (shape, size), and specialized arrays (`zeros`, `ones`, `linspace`).
2.  **[02_indexing_slicing.py](file:///d:/PYTHON%20COURSE/numpy/02_indexing_slicing.py)**: How to access and modify data using basic indexing, multi-dimensional slicing, and boolean masks.
3.  **[03_math_operations.py](file:///d:/PYTHON%20COURSE/numpy/03_math_operations.py)**: Element-wise math, aggregate functions (`sum`, `mean`), and matrix multiplication.
4.  **[04_reshaping_broadcasting.py](file:///d:/PYTHON%20COURSE/numpy/04_reshaping_broadcasting.py)**: Changing array shapes, flattening, stacking arrays, and understanding broadcasting rules.
5.  **[05_statistics_random.py](file:///d:/PYTHON%20COURSE/numpy/05_statistics_random.py)**: Generating random data, statistical analysis (`median`, `variance`), and finding unique elements.
6.  **[06_boolean_masking.py](file:///d:/PYTHON%20COURSE/numpy/06_boolean_masking.py)**: Advanced filtering, logical operators (`&`, `|`), and conditional replacement with `np.where`.
7.  **[07_real_world_practice.py](file:///d:/PYTHON%20COURSE/numpy/07_real_world_practice.py)**: A practical scenario involving temperature data analysis, normalization, and outlier detection.

## 🚀 How to use
- Install NumPy first if you haven't: `pip install numpy`
- Run each script in order to learn the concepts.
- Example: `python 01_basics.py`
